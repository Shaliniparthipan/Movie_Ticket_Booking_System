# Movie Ticket Booking System

## Project Overview

The Movie Ticket Booking System is a MySQL-based database project designed to manage movie scheduling, theater operations, ticket reservations, seat allocation, customer bookings, payments, reviews, and coupons. The project demonstrates database design and advanced SQL concepts through real-world business scenarios.

## Features

* Movie and Show Management
* Theater and Screen Management
* Customer Booking System
* Seat Reservation Tracking
* Payment Management
* Movie Reviews and Ratings
* Coupon and Discount Management
* Revenue and Booking Analytics

## SQL Concepts Used

* Joins
* Aggregate Functions
* Subqueries
* Common Table Expressions (CTEs)
* Window Functions (RANK, DENSE_RANK, ROW_NUMBER)
* Stored Procedures
* Triggers
* Indexes
* Transactions (COMMIT, ROLLBACK, SAVEPOINT)

## Database Modules

* Users
* Movies
* Theaters
* Screens
* Shows
* Seats
* Bookings
* Payments
* Reviews
* Coupons

## Project Highlights

* Designed a relational database with multiple interconnected tables.
* Implemented advanced SQL queries for business reporting and analytics.
* Built revenue tracking, booking analysis, customer insights, and movie performance reports.
* Developed Stored Procedures and Triggers to automate database operations.
